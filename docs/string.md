# 字符串工具函数

## 基础操作 (basic.ts)

### isEmpty
判断字符串是否为空。

```typescript
isEmpty(str: string): boolean
```

**参数：**
- `str`: 要检查的字符串

**返回值：**
- 如果字符串为空（空字符串、null 或 undefined），返回 true
- 否则返回 false

**示例：**
```typescript
isEmpty('') // true
isEmpty(null) // true
isEmpty(undefined) // true
isEmpty('hello') // false
```

### isNotEmpty
判断字符串是否不为空。

```typescript

```

**参数：**
- `str`: 要搜索的字符串
- `searchStr`: 要查找的子字符串
- `fromIndex`: 开始搜索的位置

**返回值：**
- 子字符串的位置，如果未找到返回 -1
- 如果输入无效，返回 -1

**示例：**
```typescript

indexOf('Hello World', '') // 0

indexOf(null, 'World') // -1
indexOf(undefined, 'World') // -1
```
