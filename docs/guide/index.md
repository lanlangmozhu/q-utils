# 指南

欢迎使用 Q-Utils！本指南将帮助你快速上手。

## 什么是 Q-Utils？

Q-Utils 是一个功能丰富的前端工具函数库，提供了日常开发中常用的各类实用工具函数。它按功能分类组织，支持按需导入，帮助开发者提高开发效率。

## 核心特性

- 🚀 **开箱即用**: 提供丰富的工具函数，覆盖日常开发中的各种场景
- 📦 **按需导入**: 支持命名空间导入和按需导入，支持 Tree Shaking
- 🔒 **类型安全**: 完整的 TypeScript 类型定义
- ⚡ **零依赖**: 不依赖任何第三方库
- 📚 **文档完善**: 详细的 API 文档和使用示例
- ✅ **测试覆盖**: 完整的单元测试覆盖
